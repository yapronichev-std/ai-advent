# FAQ: JC-Mobile SDK Android — Обзор продукта

## Что такое JC-Mobile SDK Android?

JC-Mobile SDK Android — это комплект разработчика для встраивания устройств JaCarta (токенов и смарт-карт) в мобильные приложения на платформе Android. SDK предоставляет Java-обертку над единой библиотекой PKCS #11 v2.30.

Версия: 2.5.3

## Какие устройства поддерживаются?

SDK поддерживает следующие модели токенов и смарт-карт JaCarta:
- JaCarta-2 ГОСТ
- JaCarta-2 PKI/ГОСТ
- JaCarta PKI
- JaCarta ГОСТ
- JaCarta PKI/ГОСТ

## Какие системные требования?

- **Android 6.0** или выше
- **Java Development Kit 8 (JDK 8)**
- Для сборки примеров: **JDK 11**, Android SDK, Gradle

## Какие считыватели смарт-карт поддерживаются?

JC-Mobile SDK Android работает со смарт-картами с помощью:
1. **Беспроводного интерфейса NFC** — основной способ взаимодействия на мобильных устройствах
2. **Доверенного смарт-карт ридера JCR721** — для USB-подключения

## Что такое JaCarta Service?

**JaCarta Service** — это мобильное приложение (распространяется через Google Play), которое:
- Обеспечивает взаимодействие между устройствами JaCarta и прикладным ПО
- Позволяет работать со всеми токенами и смарт-картами JaCarta
- Является обязательным компонентом — должно быть установлено на устройстве для работы SDK

## Как получить SDK?

Для получения комплекта разработчика JC-Mobile SDK Android необходимо оформить заявку на сайте разработчика (developer.aladdin-rd.ru).
