# Обзор продукта ExampleApp

## Что такое ExampleApp?

ExampleApp — это облачная платформа для управления проектами и командной работы. Позволяет создавать проекты, отслеживать задачи, обмениваться файлами и общаться с командой в реальном времени.

## Ключевые возможности

- **Управление проектами:** Канбан-доски, диаграммы Ганта, списки задач.
- **Командная работа:** Общие документы, комментарии, уведомления.
- **Интеграции:** API для подключения CRM, Telegram-боты, webhooks.
- **Отчёты:** Экспорт в PDF, Excel, настраиваемые дашборды.
- **Безопасность:** Шифрование данных, двухфакторная аутентификация, ролевая модель доступа.

## Интеграция через API

### Доступные методы

- **OAuth 2.0** — авторизация через стандартный протокол.
- **REST API** — полный доступ к проектам, задачам, пользователям.
- **Webhooks** — уведомления о событиях (создание задачи, изменение статуса).

### Как начать интеграцию

1. Получите API-ключ в Настройки → Разработчикам → API.
2. Изучите документацию: https://docs.example.com/api
3. Для OAuth используйте client_id и client_secret из настроек.
4. Для тестирования доступна песочница: https://sandbox.example.com

### Частые ошибки при интеграции

- **401 Unauthorized:** Проверьте срок действия токена. OAuth-токены живут 1 час.
- **403 Forbidden:** Убедитесь, что у API-ключа есть нужные права доступа.
- **500 Internal Server Error:** Сократите размер запроса (макс. 10 МБ для загрузки файлов).
- **Rate limit:** 100 запросов в минуту для Basic, 1000 для Pro, 10000 для Enterprise.

## Экспорт отчётов

### Как экспортировать отчёт

1. Откройте проект → Отчёты.
2. Выберите тип отчёта: Сводка, По задачам, По времени.
3. Выберите период и формат (PDF или Excel).
4. Нажмите «Скачать».

### Ошибка 500 при экспорте

**Причина:** Слишком большой объём данных для выбранного периода.

**Решение:**
1. Попробуйте экспортировать за более короткий период (неделя вместо месяца).
2. Исключите архивные проекты из отчёта.
3. Используйте фильтры, чтобы сократить объём.
4. Если ошибка повторяется для любого периода — сообщите в поддержку, указав ID проекта и период.

## Блокировка аккаунта

### Причины блокировки

- Нарушение условий использования (спам, фишинг).
- Спорная транзакция (chargeback через банк).
- Подозрительная активность (вход из необычного региона).
- Задолженность по оплате (для платных тарифов).

### Как восстановить доступ

1. Свяжитесь с поддержкой через форму на сайте.
2. Если блокировка из-за списания — предоставьте подтверждение оплаты.
3. Если блокировка из-за chargeback — необходимо отозвать претензию в банке.
4. Срок рассмотрения — до 3 рабочих дней.
