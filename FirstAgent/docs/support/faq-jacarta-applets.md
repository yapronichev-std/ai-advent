# FAQ: JC-Mobile SDK Android — Апплеты и примеры

## Какие апплеты поддерживаются?

JC-Mobile SDK Android поддерживает 4 типа апплетов:

| Апплет | Тип устройства | ГОСТ | PKI | Описание |
|--------|---------------|------|-----|----------|
| Криптотокен 2 ЭП | JaCarta-2 ГОСТ | Да (2012) | Нет | Электронная подпись ГОСТ Р 34.10-2012 |
| Laser | JaCarta PKI | Нет | Да | PKI-операции, сертификаты |
| Криптотокен | JaCarta ГОСТ | Да | Нет | Электронная подпись ГОСТ, шифрование |
| Лицензионный | — | — | — | Управление лицензиями |

## Примеры для Криптотокен 2 ЭП (GOST2_*)

| Пример | Описание |
|--------|----------|
| `GOST2_info` | Получение информации о библиотеке, ридере, слоте и апплете |
| `GOST2_initUser` | Подготовка JaCarta-2 ГОСТ к работе после инициализации в ПО "АРМ разработчика" |
| `GOST2_keyPairAndCertificate2012` | Создание/удаление ключевой пары ГОСТ Р 34.10-2012 (256 бит) и сертификата |
| `GOST2_certificateInfo` | Получение информации о сертификатах |
| `GOST2_certificateRenewal` | Создание запроса на перевыпуск сертификата |
| `GOST2_signAndVerify2012` | Создание и проверка подписи по ГОСТ Р 34.10-2012 (256 бит) |
| `GOST2_signAndVerifyExternalHash2012` | Создание и проверка подписи хэша по ГОСТ Р 34.11-2012 (256 бит) |
| `GOST2_cmsSignAndVerify2012` | Создание и проверка подписи в формате CMS signed data |
| `GOST2_changePinUser` | Смена PIN-кода пользователя |
| `GOST2_unlockPinUser` | Разблокировка PIN-кода с помощью PUK-кода |
| `GOST2_data` | Создание, поиск и удаление бинарных объектов |

## Примеры для Laser (PKI_*)

| Пример | Описание |
|--------|----------|
| `PKI_appletStatus` | Получение информации о готовности апплета к использованию |
| `PKI_init` | Инициализация апплета и выпуск сертификата |
| `PKI_info` | Получение полной информации о библиотеке, слоте и ключе |
| `PKI_pinUser` | Изменение PIN-кода пользователя |
| `PKI_certInfo` | Получение информации о сертификатах в памяти ключа |
| `PKI_reissue` | Перевыпуск сертификата |
| `PKI_delete` | Удаление сертификата с ключевой парой |
| `PKI_signAndVerify` | Создание и проверка электронной подписи |
| `PKI_CMSSignAndVerify` | Создание и проверка подписи в формате CMS |
| `PKI_data` | Запись и чтение файлов в памяти ключа |
| `PKI_pinAdmin` | Изменение PIN-кода администратора |
| `PKI_pinUserUnblock` | Сброс попыток предъявления PIN-кода пользователя |

## Примеры для Лицензионного апплета (LICENSE_*)

|--------|----------|

| `LICENSE_changeKey` | Смена ключа доступа вендора |
| `LICENSE_init` | Создание лицензий на апплете |
